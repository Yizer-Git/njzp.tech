# 图像识别界面布局优化

## 修改文件

`yolo_cropDisease_detection_vue/src/views/imgPredict/index.vue`

## 具体修改内容

### 1. 修复"重置参数"按钮样式

将第10行的按钮从文本按钮改为实心按钮:

- 移除 `text` 属性
- 添加 `size="default"` 保持一致性
- 调整样式类使其与"开始识别"按钮风格一致

### 2. 固定上传和识别结果盒子布局

修改 `.content-layout`、`.upload-pane` 和 `.result-section` 的样式:

- 设置 `.content-layout` 为固定高度,使用 `flex: 1` 撑满剩余空间
- 设置左右盒子比例为 `3:2`(左边稍大用于显示图片)
- 添加 `min-height: 0` 和 `max-height: 100%` 防止溢出
- 确保盒子高度固定,不随内容改变

### 3. 图片自适应显示

修改 `.upload-preview` 样式:

- 改为 `object-fit: contain`(保持比例,居中显示)
- 设置 `max-width: 100%` 和 `max-height: 100%`
- 添加居中对齐样式

### 4. 识别结果区域滚动

修改 `.result-section` 样式:

- 设置 `overflow-y: auto` 允许垂直滚动
- 确保内容区域固定,超出部分显示滚动条
- 保持布局稳定,不随内容增加而改变高度

### 5. 调整整体页面布局

修改 `.page-shell` 和 `.predict-card`:

- 确保页面高度计算正确
- 使用 flexbox 布局使内容区域自动填充
- 防止页面整体跳动

## 关键CSS修改点

```scss
// 重置按钮样式调整
.predict-card__reset {
  // 移除text按钮特殊样式,添加常规按钮样式
}

// 内容布局固定高度
.content-layout {
  flex: 1;
  min-height: 0;
  overflow: hidden;
}

// 上传盒子固定比例
.upload-pane {
  flex: 3;
  min-height: 0;
}

// 结果区域固定比例并可滚动
.result-section {
  flex: 2;
  overflow-y: auto;
}

// 图片自适应显示
.upload-preview {
  object-fit: contain;
  max-width: 100%;
  max-height: 100%;
}
```

## 预期效果

1. "重置参数"按钮样式与其他按钮一致
2. 上传盒子和识别结果盒子始终保持固定高度和位置
3. 图片上传后按比例缩放居中显示,不撑开盒子
4. 识别结果出现后不会导致布局跳动
5. 识别结果内容过多时右侧盒子内部出现滚动条
